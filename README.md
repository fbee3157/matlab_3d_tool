# IC价格三维走势分析工具

## 📋 工具概述

这是一个功能完整的MATLAB三维走势图生成工具，使用`plot3`函数绘制IC价格的三维走势图。工具包含以下核心功能：

- **三维走势可视化**：使用X轴（时间）、Y轴（热度/月搜索量）、Z轴（价格）
- **多IC颜色区分**：支持自定义IC型号名称和对应颜色
- **支撑/阻力位显示**：在Z轴方向绘制半透明水平面，直观显示压力和支撑位
- **时间轴缩放**：提供按"日"、"周"、"月"三种时间尺度显示数据
- **交互式配置**：通过Web界面配置参数并生成MATLAB代码

## 📁 文件说明

| 文件名 | 类型 | 功能说明 |
|--------|------|---------|
| `ic_3d_price_trend.m` | MATLAB脚本 | 完整的三维走势图生成脚本，包含模拟数据生成、图形绘制、支撑阻力位显示等功能 |
| `ic_3d_preview.html` | HTML文件 | 交互式配置界面，可在浏览器中配置参数并生成MATLAB代码 |
| `generate_matlab_code.py` | Python脚本 | 命令行工具，可生成自定义配置的MATLAB脚本 |
| `README.md` | Markdown文档 | 使用说明文档 |

## 🔧 功能特性

### 1. 三维走势图核心功能
- **X轴（时间轴）**：支持日/周/月三种时间尺度显示
- **Y轴（热度轴）**：基于月搜索量，单位为千次搜索
- **Z轴（价格轴）**：价格走势，单位为元
- **多IC并行显示**：每个IC型号使用不同颜色区分

### 2. 支撑/阻力位功能
- 在Z轴方向显示半透明水平面
- 支撑位：绿色半透明面
- 阻力位：红色半透明面
- 每个IC型号可独立设置支撑/阻力位

### 3. 时间轴缩放
- **按日显示**：显示全部数据点（每天一个点）
- **按周显示**：每周取一个数据点（最后一天）
- **按月显示**：每月取一个数据点（最后一天）
- 自动调整X轴标签和刻度

### 4. 模拟数据生成
- 自动生成IC的价格和热度数据
- 价格数据基于随机游走模型
- 热度数据包含季节性波动
- 支持自定义基准价格和波动率

### 5. 交互式配置
- 通过Web界面配置所有参数
- 实时预览配置效果
- 一键生成MATLAB代码
- 支持添加/删除IC型号

## 🚀 快速开始

### 方法一：直接运行MATLAB脚本（推荐）
1. 将`ic_3d_price_trend.m`复制到MATLAB工作目录
2. 在MATLAB命令窗口输入：
   ```matlab
   ic_3d_price_trend
   ```
3. 脚本将自动运行并显示三维走势图

### 方法二：使用交互式配置界面
1. 在浏览器中打开`ic_3d_preview.html`
2. 在配置界面中设置参数：
   - 选择时间尺度（日/周/月）
   - 设置总天数
   - 配置IC型号名称、颜色、价格、热度等
3. 点击"生成MATLAB代码"按钮
4. 代码将显示在预览区域
5. 点击"下载MATLAB脚本"保存生成的文件

### 方法三：使用Python生成工具
1. 运行Python脚本：
   ```bash
   python generate_matlab_code.py
   ```
2. 根据提示选择默认配置或输入自定义配置
3. 脚本将生成自定义的MATLAB代码文件

## ⚙️ 参数说明

### 基本参数
| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `time_scale` | 字符串 | 时间尺度：'day'/'week'/'month' | 'week' |
| `total_days` | 整数 | 总天数 | 365 |
| `start_date` | 字符串 | 起始日期 | '2026-01-01' |

### IC型号参数
| 参数 | 类型 | 说明 |
|------|------|------|
| `name` | 字符串 | IC型号名称 |
| `color` | 字符串 | 显示颜色（HEX格式） |
| `base_price` | 浮点数 | 基准价格（元） |
| `support` | 浮点数 | 支撑位价格 |
| `resistance` | 浮点数 | 阻力位价格 |
| `base_heat` | 浮点数 | 基准月搜索量 |

### 图形参数
| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `enable_grid` | 布尔值 | 是否显示网格 | true |
| `show_support_resistance` | 布尔值 | 是否显示支撑阻力位 | true |
| `line_width` | 浮点数 | 线条宽度 | 2.5 |
| `marker_size` | 整数 | 数据点标记大小 | 30 |
| `font_size` | 整数 | 字体大小 | 14 |

## 🛠️ 技术细节

### 数据生成算法
- **价格数据**：基于几何布朗运动模型
  ```matlab
  price_walk = cumsum(randn(1, total_days) * price_volatility);
  price_data = base_price * exp(price_walk);
  ```
  
- **热度数据**：包含季节性波动
  ```matlab
  seasonal_factor = 0.3 * sin(2*pi*(1:total_days)/365) + ...
                    0.2 * sin(4*pi*(1:total_days)/365) + ...
                    0.1 * sin(6*pi*(1:total_days)/365);
  heat_data = base_heat * exp(0.1 * seasonal_factor + 0.05 * heat_walk);
  ```
